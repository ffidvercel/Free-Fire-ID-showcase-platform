# **App Name**: FFID Hub

## Core Features:

- ID Listing: Allow users to list their Free Fire IDs for potential buyers to view, with details like in-game level, rare items, and price.
- Detailed ID Profiles: Create individual profile pages for each listed ID, showcasing images, videos, and other relevant information provided by the seller.
- Contact Information Display: Display seller's contact information (WhatsApp, Instagram, etc.) on the ID profile page to facilitate direct communication and transactions.
- Agreement Disclaimer: Require users to explicitly agree to a disclaimer stating that the website is for fun and entertainment purposes only before they can view ID listings.
- "Show More Details" Feature: Implement a "Show More Details" feature that reveals additional information and disclaimers about the website's purpose and safety when clicked.
- Policy Page: A dedicated page outlining the website's policies, emphasizing its nature as a platform for fun and that calls received may also be for fun.

## Style Guidelines:

- Primary color: Deep indigo (#4B0082), chosen to subtly evoke the sophistication of gaming and high-value transactions.
- Background color: Very light grayish-blue (#F0F8FF).
- Accent color: Saturated violet (#8A2BE2) to highlight key interactive elements and CTAs.
- Body font: 'Inter', sans-serif, for a modern and readable interface. Headline font: 'Space Grotesk', sans-serif, providing a tech-focused and eye-catching contrast.
- Use high-quality icons related to gaming accounts, such as level badges, character skins, and item collections.
- Clean and organized layout with clear sections for ID listings, profiles, and contact information.
- Subtle transitions and animations to enhance user experience and provide visual feedback on interactions.
